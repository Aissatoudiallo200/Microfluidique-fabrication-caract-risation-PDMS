# 🔬 Caractérisation de microcanaux PDMS – Microfluidique  

## 📌 Objectif  
Fabrication et caractérisation expérimentale d’un microcanal PDMS en régime laminaire, avec modélisation des données sous Python.

## 🧪 Travail réalisé  
- Fabrication de microcanaux PDMS (moulage, dégazage, plasma bonding)  
- Mesures pression–débit avec pompe Elveflow OB1  
- Régression linéaire (ΔP vs Q) pour extraire la résistance hydraulique  
- Estimation du rayon équivalent via la loi de Poiseuille  
- Validation du régime laminaire (Re < 1)

## 📊 Outils  
Microfluidique · Mécanique des fluides · Python · Analyse et modélisation de données
